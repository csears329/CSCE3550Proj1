Open main.py, run it then you can go to http://localhost:8080/register on your browser and make an account. Then a password will be generated. 
If you go to http://localhost:8080/admin/users you can see the lists of users, when their account was created, the last time they logged in, their user ID, and the hashed password. 
The project doesn't require a login page so the last login will remain empty.
